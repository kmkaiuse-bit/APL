# 功能更新總結 - 展開查看 Prompt

## 🎉 新功能概述

您的 AI 零售數據分析工具箱現在支援 **在網頁上直接查看完整 Prompt 內容**！

---

## ✨ 新增功能詳情

### 1. 查看完整指令按鈕

每個模組卡片現在包含：

```
┌─────────────────────────────────┐
│ MODULE 01 🔍 數據體檢            │
│ 排除數據錯誤與雜訊...            │
│                                  │
│ [👁️ 查看完整指令 ▼]  ← 新增！   │
│                                  │
│ [📋 複製分析指令]                │
└─────────────────────────────────┘
```

### 2. 展開/收起動畫效果

- **展開時**：
  - 順暢向下展開（0.4 秒動畫）
  - 按鈕文字變為「🔼 收起指令 ▲」
  - Prompt 內容顯示在淺灰色背景框中

- **收起時**：
  - 順暢向上收起
  - 按鈕恢復為「👁️ 查看完整指令 ▼」

### 3. Prompt 內容展示

```
╔════════════════════════════════╗
║ 請讀取上傳的 CSV 數據...       ║
║                                 ║
║ 具體檢查項目：                 ║
║ 1. 是否有空值或缺失數據        ║
║ 2. 數值範圍是否合理...         ║
║ 3. 是否有重複的記錄            ║
║ 4. 數據類型是否一致            ║
║                                 ║
║ 請以表格形式列出...            ║
╚════════════════════════════════╝
```

- **淺灰色背景**：易於區分內容
- **等寬字體**：保持格式整齊
- **可滾動**：內容超過 600px 時自動滾動
- **完整格式**：保留所有換行和縮進

## 💻 技術實現

### CSS 動畫
```css
.prompt-container {
    max-height: 0;
    overflow: hidden;
    transition: max-height 0.4s ease, opacity 0.3s ease;
}

.prompt-container.expanded {
    max-height: 600px;
    opacity: 1;
}
```

### JavaScript 邏輯
```javascript
function togglePrompt(moduleId) {
    const container = document.getElementById(`prompt-${moduleId}`);
    const button = document.getElementById(`toggle-btn-${moduleId}`);

    if (container.classList.contains('expanded')) {
        // 收起
        container.classList.remove('expanded');
        button.innerHTML = '👁️ 查看完整指令 ▼';
    } else {
        // 展開
        container.classList.add('expanded');
        button.innerHTML = '🔼 收起指令 ▲';
    }
}
```

## 🚀 如何測試

### 快速測試步驟

1. **打開應用**
   ```bash
   雙擊 index.html
   ```

2. **測試展開功能**
   - 點擊任一「查看完整指令」按鈕
   - 觀察 Prompt 內容是否順暢展開
   - 檢查按鈕文字是否變為「收起指令」

3. **測試收起功能**
   - 點擊「收起指令」按鈕
   - 觀察 Prompt 內容是否順暢收起
   - 檢查按鈕文字是否恢復

4. **測試多模組展開**
   - 同時展開 2-3 個模組
   - 確認各模組互不影響

5. **測試複製功能**
   - 在展開狀態下點擊「複製指令」
   - 確認 Toast 通知顯示
   - 貼到記事本確認內容正確

6. **測試鍵盤快捷鍵**
   - 按數字鍵 1-5
   - 確認仍然可以快速複製
