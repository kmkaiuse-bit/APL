# 快速開始指南

## 🚀 5 分鐘快速上手

### 第 1 步：打開應用
```
雙擊 index.html 檔案
```
應用會在瀏覽器中自動打開。

---

### 第 2 步：查看完整指令（新功能！）

在任一模組卡片上：

1. **點擊「👁️ 查看完整指令 ▼」按鈕**
   ```
   ┌─────────────────────────────┐
   │ MODULE 01 🔍 數據體檢        │
   │ 排除數據錯誤與雜訊...        │
   │                              │
   │ [👁️ 查看完整指令 ▼]         │  ← 點擊這裡
   └─────────────────────────────┘
   ```

2. **Prompt 內容展開**
   ```
   ┌─────────────────────────────┐
   │ MODULE 01 🔍 數據體檢        │
   │ 排除數據錯誤與雜訊...        │
   │                              │
   │ [🔼 收起指令 ▲]              │
   │                              │
   │ ╔═══════════════════════╗   │
   │ ║ 請讀取上傳的 CSV 數據...║   │  ← 展開的內容
   │ ║ 具體檢查項目：         ║   │
   │ ║ 1. 是否有空值...       ║   │
   │ ╚═══════════════════════╝   │
   │                              │
   │ [📋 複製分析指令]           │
   └─────────────────────────────┘
   ```

3. **再次點擊收起**
   - 點擊「🔼 收起指令 ▲」按鈕
   - 內容會收起隱藏

---

### 第 3 步：複製指令

1. **點擊「📋 複製分析指令」按鈕**
   - 指令會自動複製到剪貼簿
   - 右上角顯示「✓ 已複製！請貼入 AI 視窗」通知

2. **使用鍵盤快捷鍵（更快！）**
   ```
   按鍵 1 → 複製「數據體檢」指令
   按鍵 2 → 複製「規律挖掘」指令
   按鍵 3 → 複製「銷量預測」指令
   按鍵 4 → 複製「專業圖表」指令
   按鍵 5 → 複製「職涯說服」指令
   ```

---

### 第 4 步：貼入 AI 工具

1. 打開 ChatGPT 或 Gemini
2. 按 **Ctrl+V** (Windows) 或 **⌘+V** (Mac) 貼入指令
3. 上傳 `professional_retail_v2_zh.csv` 數據檔案
4. 按下發送，開始分析！

## ❓ 常見問題

### Q1: 點擊複製按鈕沒反應？
**A**: 工具會自動顯示「手動複製視窗」：
1. 視窗中的文字已自動選取
2. 按 **Ctrl+C** (Windows) 或 **⌘+C** (Mac) 複製
3. 關閉視窗後貼入 AI 工具

### Q2: 可以同時展開所有模組嗎？
**A**: 可以！每個模組的展開/收起互不影響。

### Q3: 展開的內容太長怎麼辦？
**A**: 展開區域最高 600px，超出部分可以滾動查看。

### Q4: 鍵盤快捷鍵不工作？
**A**: 確保：
- 瀏覽器視窗處於活動狀態
- 沒有其他輸入框獲得焦點
- 直接按數字鍵 1-5（不需要按 Ctrl 或其他組合鍵）

## 🎓 展位演示最佳實踐

### ✅ 建議做法
- 事先準備好 ChatGPT/Gemini 帳號並登入
- 提前上傳數據檔案到 AI 工具（可節省時間）
- 按 F11 進入全螢幕模式，更專業
- 使用鍵盤快捷鍵，展示效率
- 展開 1-2 個模組的 Prompt，展示專業性

### ❌ 避免做法
- 不要在演示時才開始註冊 AI 帳號
- 不要逐字朗讀整個 Prompt（太耗時）
- 不要演示所有 5 個模組（選擇 3-4 個即可）
- 不要在網路不穩定時演示（建議使用本地 CSV）
