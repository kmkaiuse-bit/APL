# HKIT ApL：AI 商業數據分析指揮中心

> **一鍵複製專業分析指令 × 10分鐘展示數據分析能力**

這是一個為 HKIT ApL 課程設計的教育展位演示工具，幫助學生在家長面前展示如何運用 AI 進行專業的商業數據分析。

## 🚀 快速開始

### 最簡單的使用方式
1. **雙擊** `index.html` 檔案
2. 檔案會在瀏覽器中自動打開
3. **點擊「👁️ 查看完整指令」** 按鈕可預覽 Prompt 內容
4. **點擊「📋 複製分析指令」** 按鈕即可複製指令
5. 貼入 ChatGPT 或 Gemini，上傳 `professional_retail_v2_zh.csv` 數據檔案
6. 完成專業數據分析！

### 支援的瀏覽器
- ✅ Google Chrome（推薦）
- ✅ Microsoft Edge
- ✅ Mozilla Firefox
- ✅ Safari

## 📊 五大分析模組

### 🔍 MODULE 01：數據體檢
**作用**：排除數據錯誤與雜訊，確保生意分析準確可靠

**包含內容**：
- 空值與缺失數據檢查
- 數值範圍合理性驗證
- 重複記錄識別
- 數據類型一致性檢查

---

### 📊 MODULE 02：規律挖掘
**作用**：發現天氣與時間如何影響生意，找出賺錢的隱藏規律

**包含內容**：
- 氣溫與銷量的相關係數分析
- 產品敏感度排序
- 正負相關產品識別
- 商業洞察提煉

---

### 🎯 MODULE 03：銷量預測
**作用**：科學預知未來產品銷量，幫你精準決定進貨量

**包含內容**：
- 多元線性回歸模型建立
- 預測公式生成
- R² 值計算
- 95% 信賴區間分析
- 進貨建議

---

### 📈 MODULE 04：專業圖表
**作用**：將枯燥數據轉成專業圖表，讓商業表現一眼看清

**包含內容**：
- 考試週 vs 非考試週銷量對比柱狀圖
- 氣溫與凍飲銷量趨勢分析圖
- 商業簡報級視覺設計

---

### 💼 MODULE 05：職涯說服
**作用**：學會用數據說服老闆決策，鎖定未來的 AI 高薪職位

**包含內容**：
- 1 分鐘投資者簡報劇本
- 核心發現與商業影響分析
- 具體行動方案
- 2026 年 3 個熱門高薪職位及起薪點

## 🎯 展位部署步驟

### 方法一：本地執行（推薦）
1. 將 `Demo` 資料夾複製到展示用電腦
2. 雙擊 `index.html` 開啟
3. 按 `F11` 進入全螢幕模式
4. 開始演示！

### 方法二：USB 隨身碟部署
1. 將整個 `Demo` 資料夾複製到 USB 隨身碟
2. 在展位電腦上從隨身碟開啟 `index.html`
3. 完全離線可用，無需網路

### 方法三：網頁伺服器部署（選用）
如需透過網路分享：

```bash
# 使用 Python 內建伺服器
cd Demo
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

然後訪問 `http://localhost:8000`

## 🛠️ 故障排除

### 問題：點擊按鈕後沒有反應

**原因**：瀏覽器安全政策可能阻擋自動複製功能

**解決方案**：
1. 工具會自動顯示「手動複製視窗」
2. 視窗中的文字已自動選取
3. 按 `Ctrl+C` (Windows) 或 `⌘+C` (Mac) 手動複製
4. 關閉視窗後貼入 AI 工具即可

---

### 問題：在舊版瀏覽器中顯示異常

**解決方案**：
- 更新瀏覽器至最新版本
- 使用 Chrome 或 Edge（推薦）

---

### 問題：手機上點擊按鈕困難

**解決方案**：
- 按鈕最小尺寸已設為 44x44px（符合觸控標準）
- 使用橫向模式以獲得更大的點擊區域
- 使用放大手勢（雙指縮放）
