### 複製功能測試（三層 Fallback）

#### 測試 1：Clipboard API（現代瀏覽器）
- [ ] 在 Chrome/Edge 中點擊「數據體檢」卡片
- [ ] 右上角出現綠色 Toast 通知「已複製！請貼入 AI 視窗」
- [ ] Toast 在 3 秒後自動消失
- [ ] 貼到記事本/文字編輯器，確認內容正確

#### 測試 2：execCommand Fallback
- [ ] 在較舊的瀏覽器或 `file://` 協議下測試
- [ ] 如果 Clipboard API 失敗，execCommand 應自動啟用
- [ ] Toast 仍然正常顯示

#### 測試 3：手動複製模態視窗（終極備用）
- [ ] 在完全不支援的環境下（或手動觸發失敗）

- [ ] 模態視窗中的 Textarea 顯示完整 Prompt 內容
- [ ] 文字已自動選取（高亮顯示）
- [ ] 按 Ctrl+C (Windows) 或 ⌘+C (Mac) 可成功複製
- [ ] 點擊「我已複製」或「關閉」按鈕，模態視窗關閉
- [ ] 點擊背景半透明區域，模態視窗也應關閉
