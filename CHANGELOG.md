# 更新日誌

## 版本 1.1.0 (2026-02-24)

### ✨ 新增功能

#### 展開查看 Prompt 功能
- **新增「查看完整指令」按鈕**
  - 每個模組卡片增加一個獨立的「👁️ 查看完整指令」按鈕
  - 點擊後可在網頁上展開查看完整的 Prompt 內容
  - 再次點擊可收起內容

- **視覺設計**
  - 展開的 Prompt 內容使用淺灰色背景框顯示
  - 保持 monospace 字體，確保格式正確
  - 添加順暢的展開/收起動畫效果（0.4 秒）
  - 按鈕圖標會隨狀態變化（▼ ↔ ▲）

- **用戶體驗**
  - 支援多個模組同時展開
  - 展開內容最高 600px，超出部分可滾動
  - 按鈕設計：透明背景 + 藍色邊框，懸停時填充藍色

### 🎯 使用場景

1. **向家長展示專業性**
   - 展開 Prompt 內容，展示指令的專業性和完整性
   - 讓家長看到學生使用的 AI 分析工具的真實內容

2. **複製前預覽**
   - 在複製前先查看完整指令
   - 確認內容符合需求後再複製

3. **教學演示**
   - 教師可以逐一展開模組，向學生講解每個分析步驟
   - 適合課堂教學和工作坊使用

### 📝 文檔更新

- 更新 README.md，添加「查看完整指令」功能說明
- 更新 TEST_CHECKLIST.md，添加展開功能測試項目
- 更新頁面 Footer 的使用說明
